How to Play?
Rule Sets
  • The Snake is hungry and you have to feed it.The ultimate aim is to Collect the food which inturns grow the sanke and increase the score.
  • You can move the snake using the
    • KeyBoard Controls
    • Onscreen Joypad Controls
    • Swipe Controls
  • It has Three Modes :
    Classic
    In this mode there will be no maze or outside constraints. For example, if you go right you will come from left.
    Game over Rules:
    There is only one condition that leads to the end of the game:
    The sanke collides itself.
    Modern
    This has five levels namely: Box, Tunnel, Garden, Apartment. Each of it has its own preset Maze.
    Game over Rules:
    1. The snake collides itself.
    2. The sanke collides the maze.
    Campaign
    This is the mode which connects all the Modern modes.There will be a target for each level in the Campaign mode. If you reach the target you will be upgraded to the next level.
    Game over Rules:
    1. The snake collides itself.
    2. The sanke collides the maze.
    Safepoint
    There will be a safepoint score for each level in the Campaign mode. If you reach that and you encounter a collision you will not be sent to the first level. Or else the game will begin from the first level.
